Figura Fórmula da Área Fórmula do Perímetro Exemplo com Medidas
Quadrado A = lado² P = 4 × lado lado = 5 → A = 25, P = 20
Retângulo A = base × altura P = 2(base + altura) base = 8, altura = 3 → A = 24, P = 22
Círculo A = π × r² P = 2 × π × r r = 4 → A ≈ 50,27, P ≈ 25,13
Triângulo A = (base × altura) / 2 P = a + b + c base = 6, altura = 4, lados = 6,5,5 → A = 12, P = 16
1Todas as fórmulas assumem unidades métricas (cm, m, etc). Valores aproximados com π ≈ 3,14.
